#Hito 1 - Script que consume la API de GitHub

## Descripción 

Script en Python que consume la API de Github, procesa los datos de un usuario y los guarda en JSON.

## Instalación
1. Clona el repositorio
2. Crea un entorno virtual : `python3 -m venv .venv`
3. Actívalo : `source venv/bin/activate`
4. Instala dependecias : `pip install -r requirements.txt`

## Configuración

1. Crea un fichero `.env` con :
BASE_URL=https://api.github.com
TOKEN=tu_token_de_github
USUARIO=nombre_de_usuario

2. USO = python3 main.py
